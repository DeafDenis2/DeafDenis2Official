System Informer

A powerful system utility designed to provide detailed information about system resources, processes, and performance metrics. Essential for analyzing running processes, monitoring system behavior, and troubleshooting performance issues.

See what's happening under the hood

DnSpy

A comprehensive tool to reverse engineer .NET assemblies. It can be used to debug, decompile, and edit assemblies even without source code available, making it invaluable for .NET application analysis and security research.

Decompile, debug, discover

x64dbg

An open-source x64/x32 debugger for Windows. Used for dynamic analysis of binaries, it's a popular alternative to commercial debuggers in the reverse engineering community, featuring a rich plugin ecosystem.

Debug with precision and power
